📋 Toplanan Veriler

Pawzy, hizmetlerini sunabilmek için aşağıdaki verileri işler:

📱 Cihazda Saklanan Veriler

  • • Evcil hayvan adı, türü, yaşı, kilosu, cinsiyeti
  • • İlaç adı, dozajı, hatırlatma saatleri
  • • Sohbet geçmişi (AI asistan ile yapılan konuşmalar)

🤖 AI Özelliği — DeepSeek API

Pawzy AI özelliği kullanıldığında DeepSeek API'ye gönderilir:

  • • Sohbet mesajlarınız
  • • Evcil hayvanınızın türü, yaşı, kilosu, cinsiyeti (isim anonimleştirilir)

İlk kullanımda onayınız alınır, sadece izin verdiğinizde veri gönderilir.

✨ Günlük İpucu (Premium)

Pet türü ve yaş bilgisi DeepSeek API'ye anonim olarak gönderilir.

🔒 Veri Paylaşımı

Pawzy, verilerinizi şu durumlar dışında üçüncü taraflarla paylaşmaz:

  • 🤖 AI özelliğinde DeepSeek API'ye veri gönderilir
  • ☁️ iCloud yedekleme açıksa veriler Apple iCloud'da şifreli saklanır

✅ Hiçbir veriniz reklam, analitik veya başka amaçlarla paylaşılmaz.

☁️ iCloud

İsteğe bağlı iCloud Drive yedekleme. Etkinleştirilmezse veriler sadece cihazınızda kalır.

🛡️ Veri Güvenliği

Cihazdaki veriler SwiftData ile yerel saklanır. iCloud yedekleri şifrelidir.

✅ Kullanıcı Hakları

  • ✓ AI özelliğini kullanmama hakkınız var
  • ✓ iCloud yedeklemeyi istediğiniz zaman kapatabilirsiniz
  • ✓ Uygulamayı silerek tüm yerel verileri kaldırabilirsiniz

⚠️ Sorumluluk Reddi

Pawzy tıbbi tavsiye vermez. İlaç hatırlatıcıları ve AI yanıtları bilgilendirme amaçlıdır. Evcil hayvanınızın sağlığı için veterinerinize danışın.

📋 Data Collection

Pawzy processes the following data:

📱 Data Stored on Device

  • • Pet name, breed, age, weight, gender
  • • Medication name, dosage, reminder times
  • • Chat history (AI assistant conversations)

🤖 AI Feature — DeepSeek API

When using Pawzy AI, sent to DeepSeek API:

  • • Your chat messages
  • • Pet breed, age, weight, gender (name is anonymized)

Consent required on first use. Data sent only with permission.

✨ Daily Tip (Premium)

Anonymized pet breed and age sent to DeepSeek API.

🔒 Data Sharing

Pawzy does not share your data except:

  • 🤖 AI feature sends data to DeepSeek API
  • ☁️ iCloud backup enabled → encrypted on Apple iCloud

✅ No data shared for advertising, analytics, or other purposes.

☁️ iCloud

Optional iCloud Drive backup. Data stays on device unless enabled.

🛡️ Data Security

Device data stored locally with SwiftData. iCloud backups are encrypted.

✅ User Rights

  • ✓ Right to not use AI feature
  • ✓ Disable iCloud backup anytime
  • ✓ Delete app to remove all local data

⚠️ Disclaimer

Pawzy does not provide medical advice. Reminders and AI responses are for informational purposes only. Always consult your veterinarian.
